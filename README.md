# Bazar.in 🛒

Bazar.in is a full-stack e-commerce web platform designed to provide a smooth and modern online shopping experience.
The project includes a frontend user interface, backend APIs, database integration, and a secure online payment system.

---

## Features

* User authentication and account management
* Product catalog with categories
* Shopping cart system
* Order management
* AI shopping assistant (style adviser robot icon)
* Secure online payment integration
* Responsive frontend interface

---

## Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* SQL (PostgreSQL / MySQL structure)

### Payment Gateway

* Razorpay integration for secure online payments

## Payment System

The platform integrates **Razorpay** to handle secure online payments.
The backend verifies payment signatures and stores transaction details in the database.
